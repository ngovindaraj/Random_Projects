This file describes the contents of each file included in the submission folder

1. report.md        :   Final Report containing analysis

2. audit.py         : code for auditing and cleaning the original dataset

3. schema.py        : xml schema that needs to be validated

4. osm_to_csv.py    : code for converting the cleaned osm file to csv, and validate the schema

5. sample.py  	    : code for creating a sample from the original osm dataset; used for schema validation and preliminary analysis

6. san-francisco-sample.osm : contains sample data of the map region used (1-10 mb in size)

7. sql_db.py  : code for creating a database of csv files
